# 🎬 电影知识图谱系统

基于MovieLens数据集的电影关系网络探索与可视化平台

## 📖 项目简介

电影知识图谱系统是一个基于Neo4j图数据库的电影关系网络探索平台。系统使用经典的MovieLens ml-latest-small数据集，通过知识图谱技术展示电影、用户、评分和标签之间的复杂关系，提供交互式的数据探索和可视化体验。

## ✨ 核心功能

### 🎥 电影列表

- **浏览完整电影库**：分页浏览9742部电影数据
- **智能搜索**：支持按电影标题快速搜索
- **详细信息**：查看每部电影的基本信息（标题、年份、类型）
- **快速跳转**：一键跳转到电影的关系网络视图

### 🔗 关系网络图

- **交互式可视化**：动态展示电影关系网络
- **多度关系查询**：支持1-3度关系深度探索
- **节点数量控制**：可调节显示节点数量（50-300个）
- **关系类型展示**：显示电影-类型、用户-评分等关系
- **点击探索**：点击节点展开更多关系连接

### 🌐 六度空间查询

- **路径发现**：查找任意两个用户之间的最短关系路径
- **实时搜索**：快速搜索用户并建立连接
- **可视化路径**：图形化展示关系路径
- **度数显示**：展示路径度数和完整连接链条

### 🤖 智能推荐

- **个性化推荐**：基于用户历史评分的智能推荐
- **多种推荐策略**：
  - 基于电影类型偏好
  - 基于相似用户行为
  - 基于知识图谱的关系推理

## 📊 数据集详情

### 数据来源

本项目使用 **MovieLens ml-latest-small** 数据集，由明尼苏达大学的GroupLens研究组提供和维护。

### 数据规模

| 数据类型 | 数量      | 说明                     |
| :------- | :-------- | :----------------------- |
| 电影     | 9,742部   | 包含标题、类型等信息     |
| 用户     | 610位     | 所有用户至少评分20部电影 |
| 评分     | 100,836条 | 评分范围：0.5-5.0星      |
| 标签     | 3,683条   | 用户生成的电影标签       |

### 时间跨度

- **起始时间**：1996年3月29日
- **结束时间**：2018年9月24日
- **数据更新**：数据集包含22年的用户行为数据

### 电影类型

系统包含19种电影类型：动作、冒险、动画、儿童、喜剧、犯罪、纪录片、剧情、奇幻、黑色电影、恐怖、音乐、神秘、爱情、科幻、惊悚、战争、西部

### 数据文件

```
ml-latest-small/
├── movies.csv         # 电影基本信息
├── ratings.csv        # 用户评分记录
├── tags.csv           # 用户标签数据
└── links.csv          # 外部链接（IMDB/TMDB）
```

## 🛠️ 技术架构

### 后端技术栈

- **Python 3.8+** - 主要编程语言
- **FastAPI** - 高性能Web框架
- **Neo4j** - 图数据库（社区版/企业版）
- **py2neo** - Neo4j Python驱动
- **python-dotenv** - 环境变量管理
- **Uvicorn** - ASGI服务器

### 前端技术栈

- **Vue 3** - 前端框架
- **Element Plus** - UI组件库
- **ECharts** - 数据可视化库
- **Vue Router** - 路由管理
- **Axios** - HTTP客户端
- **D3.js** - 网络图可视化

### 数据层

- **Neo4j图数据库** - 主数据存储
- **Cypher查询语言** - 图数据查询
- **APOC插件** - 高级图算法
